Decompile A Java Class File

先给出一段简单的代码如下：

public class Test {
    public static void main(String[] args) {
        int j = 0;
        for (int i = 0; i < 7 ; i ++) {
            j = j++;
        }
        System.out.println(j);
    }
}

程序输出为0，有点不解，Google后发现是“Java中间缓存变量机制”，还是不清不楚。所以使用javap对编译后的class文件进行分析。

指令为：

javac Test.java
javap -c Test

得到如下代码：
